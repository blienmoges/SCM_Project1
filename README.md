# Student Portal Management System (SCM Mini Project)

## Team
Team Size: 6

## Project Summary
A simple Student Portal system focused on Software Configuration Management (SCM) practices using Git/GitHub.
The system contains:
- Login/Authentication page
- Main Dashboard (modules list)
- Core action: Course Registration (Add/Drop)
- JSON files for simple data storage

## Repository Structure
- /docs      -> SCM documents (SCMP, CI register, CRs, baselines, audit, etc.)
- /src       -> Source code (HTML/CSS/JS) + JSON data
- /tests     -> Test cases
- /releases  -> Release notes

## How to Run
1. Open `src/login.html` in a browser.
2. Login with sample accounts from `src/data/students.json`.
3. After login, you will be redirected to `dashboard.html`.
